Accepted file structure

Each file must contain a date, a description or reference, and either one amount column or debit/credit columns. BookPoint detects common headers such as Date, Transaction Date, Posting Date, Value Date, Description, Particulars, Remarks, Payee, Supplier, Account Title, Reference No, Voucher No, Invoice No, Debit, Credit, Withdrawal, Deposit, Amount, Payment Amount, Invoice Amount, System Amount, and Balance.

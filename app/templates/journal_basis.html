How this works

  1. Upload your January–April 2026 journal entries here. Any file with Account Code, Account Title, Debit, and Credit columns is accepted.
  2. The system reads every line, groups them by description, and learns which accounts appear together — including Input VAT splits.
  3. When you upload a new bank statement under Bank Upload, the system matches transaction descriptions to your learned patterns and built-in rules.
  4. If a match is found, a journal entry is automatically generated in Draft status. Go to Journal Entries to review and approve each one before it is posted.
